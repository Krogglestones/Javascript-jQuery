Really Simple Accordion Explaining the data- Attribute

What is the data- attribute in HTML?
The data- attribute is used to store custom data private to the page or application. It gives developers the ability to embed custom data on ALL HTML elements.
How is a data- attribute structured?
  1. The attribute name should not contain any uppercase letters, and must be at least one character long after the prefix "data-"
  2. The attribute value can be any string
Is it supported by all browsers?
Yes view information at https://caniuse.com/#search=data-
Are there any known issues with data- ?
There are two known issues.
  • Android 2.3 cannot read data-* properties from select elements.
  • Safari 10 and below can return undefined on properties of dataset in some cases.
